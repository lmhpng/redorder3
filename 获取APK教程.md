## 📦 如何免费获取APK（约5分钟）

### 第一步：注册GitHub账号
1. 打开 https://github.com
2. 点击右上角 **Sign up**，注册免费账号
3. 完成邮箱验证

---

### 第二步：创建新仓库
1. 登录后点击右上角 **+** → **New repository**
2. Repository name 填写：`VoiceRecorder`
3. 选择 **Public**（公开）
4. 点击 **Create repository**

---

### 第三步：上传项目文件
1. 在新建的仓库页面，点击 **uploading an existing file**
2. 把 `RecorderApp` 文件夹里的**所有文件和文件夹**拖拽进去
   - 注意：要保持文件夹结构不变
3. 拖拽完成后，滚动到底部点击 **Commit changes**

---

### 第四步：等待自动构建（约3-5分钟）
1. 点击仓库顶部的 **Actions** 标签
2. 你会看到一个正在运行的任务 "Build Android APK"
3. 等待绿色勾出现 ✅

---

### 第五步：下载APK
1. 点击已完成的任务
2. 滚动到页面底部找到 **Artifacts**
3. 点击 **VoiceRecorder-APK** 下载
4. 解压下载的 zip 文件，里面就是 `app-release.apk`

---

### 第六步：安装到手机
1. 将 APK 文件传输到手机（微信/QQ/数据线均可）
2. 手机打开 APK 文件
3. 如提示"来自未知来源"，在设置中允许安装即可
4. 安装完成，打开"语音录音机" App！

---

## ⚠️ 注意事项

- **系统要求**：Android 8.0 (API 26) 及以上
- **录音权限**：首次打开需授权麦克风权限
- **AI总结**：使用本地算法分析，无需联网，无需API Key
- **录音文件**：保存在App私有存储，卸载后删除

---

## 🔧 遇到问题？

如果 GitHub Actions 构建失败：
1. 点击失败的任务查看错误日志
2. 常见原因：文件上传不完整（检查 `.github/workflows/build.yml` 是否存在）
